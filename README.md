# 🗳️ Quiz: Our Brand is Crisis (Bolivia 2002)

Este es un proyecto educativo interactivo desarrollado en **Python** que analiza las estrategias de marketing político y comunicación presentadas en el documental _"Our Brand is Crisis"_.

El cuestionario está diseñado para estudiantes de ciencias sociales, comunicación o política que deseen poner a prueba su comprensión sobre el caso de Gonzalo Sánchez de Lozada y la consultoría de James Carville en Bolivia.

## 🚀 Características

- **20 Preguntas Críticas:** Basadas directamente en los eventos del documental.
- **Interfaz Web Moderna:** Desarrollado con [Streamlit](https://streamlit.io/).
- **Sistema de Feedback:** Resultados inmediatos y métricas de desempeño.
- **Botón de Reinicio:** Permite volver a intentar el reto para mejorar el aprendizaje.

## 🛠️ Tecnologías Utilizadas

- **Lenguaje:** Python 3.x
- **Framework:** Streamlit
- **Lógica de Datos:** JSON para el manejo de preguntas y respuestas.
